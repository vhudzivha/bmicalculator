# BMI Calculator Web Application

This project is a BMI (Body Mass Index) calculator developed using:

* HTML
* CSS
* PHP
* XAMPP
* Visual Studio Code

## Features

* User inputs first name, last name, age, weight and height.
* BMI is calculated automatically.
* BMI category is displayed (Underweight, Normal weight, Overweight, Obese).

## How to Run

1. Install XAMPP
2. Place the project folder inside `htdocs`
3. Start Apache
4. Open the browser and go to:

```
http://localhost/bmicalculator/
```
